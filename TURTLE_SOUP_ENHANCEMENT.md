## 🔧 技术实现

### 问答算法逻辑
```typescript
// 1. 问题有效性检查
- 检测是否包含"为什么"、"是谁"等非yes/no问题
- 提示用户使用正确的提问方式

// 2. 关键词匹配分析
- 提取汤底和汤面关键词
- 计算问题与内容的关联度
- 判断问题是否与故事相关

// 3. 智能回答生成
- 高关联度 + 否定词 → "否"
- 高关联度 + 肯定词 → "是"
- 低关联度 → "无关" + 引导提示
```

- 实时更新问答统计
