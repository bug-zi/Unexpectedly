# 腾讯云 CDN 配置完整指南

## 前置条件

- ✅ Vercel 部署已完成
- ✅ 拥有自己的域名（如 `example.com`）
- ✅ 腾讯云账号已实名认证

## 📋 配置步骤

### 步骤 1：确定要使用的域名

假设你的域名是 `example.com`，你可以选择：

**选项 A：使用子域名（推荐）**
- CDN 加速域名：`wanwan.example.com`
- 用户访问：`https://wanwan.example.com`

**选项 B：使用主域名**
- CDN 加速域名：`example.com` 或 `www.example.com`
- 用户访问：`https://www.example.com`

### 步骤 2：在腾讯云 CDN 控制台添加域名

1. **登录腾讯云控制台**
   - 访问：https://console.cloud.tencent.com/cdn

2. **添加域名**
   - 点击左侧菜单 **"域名管理"**
   - 点击 **"新增域名"** 按钮

3. **填写域名配置**

   | 配置项 | 填写内容 | 说明 |
   |--------|----------|------|
   | **加速域名** | `wanwan.example.com` | 替换为你的实际域名 |
   | **业务类型** | **静态加速** | 适合 Vercel 静态网站 |
   | **源站类型** | **源站域名** | Vercel 域名作为源站 |
   | **源站地址** | `wanwan-xiangdao.vercel.app` | 你的 Vercel 域名 |
   | **回源协议** | **HTTPS** | Vercel 强制 HTTPS |
   | **回源 Host** | `wanwan-xiangdao.vercel.app` | 与源站地址一致 |

4. **服务区域配置**
   - 建议选择：**中国大陆**（如果主要用户在国内）
   - 或选择：**全球加速**（如果需要海外访问）

5. **点击 "提交"**

### 步骤 3：配置 DNS 解析

腾讯云 CDN 会分配一个 **CNAME** 记录，格式类似：
```
wanwan.example.com.cdn.dnsv1.com
```

1. **进入你的域名 DNS 管理页面**
   - 如果域名在腾讯云：https://console.cloud.tencent.com/cns
   - 如果域名在其他服务商：进入相应的 DNS 管理后台

2. **添加 DNS 记录**

   | 记录类型 | 主机记录 | 记录值 |
   |----------|----------|--------|
   | **CNAME** | `wanwan` | `wanwan.example.com.cdn.dnsv1.com`（替换为实际 CNAME） |

3. **保存 DNS 配置**

### 步骤 4：配置 HTTPS 证书

1. **在 CDN 控制台配置证书**
   - 进入域名详情页
   - 点击 **"HTTPS 配置"**
   - 选择 **"云解析 DNS"** 或 **"自有证书"**
   - 如果使用腾讯云 DNS，可以申请免费证书
   - 开启 **"强制 HTTPS"**

2. **或使用 Let's Encrypt 免费证书**
   - 选择 **"自有证书"**
   - 上传证书文件和私钥

### 步骤 5：配置缓存策略（可选但推荐）

1. **进入域名配置 → 缓存配置**
2. **添加缓存规则**

   | 规则类型 | 目录/文件 | 缓存时间 |
   |----------|-----------|----------|
   | **文件类型** | `.js,.css,.png,.jpg,.jpeg,.webp,.svg,.woff,.woff2` | **30 天** |
   | **目录** | `/assets/*` | **30 天** |
   | **首页** | `/index.html` | **不缓存** 或 **3 秒** |

3. **节点缓存键配置**
   - 建议忽略参数：开启
   - 这可以提升缓存命中率

### 步骤 6：等待 CDN 生效

- **DNS 生效时间**：通常 5-30 分钟
- **CDN 全球节点生效**：最多 2 小时

### 步骤 7：验证 CDN 配置

1. **检查 DNS 解析**
   ```bash
   # Windows
   nslookup wanwan.example.com

   # 应该返回腾讯云 CDN 的 IP 地址
   ```

2. **访问测试**
   - 打开浏览器访问：`https://wanwan.example.com`
   - 检查页面是否正常加载

3. **检查 CDN 命中率**
   - 在 CDN 控制台 → **监控统计** → **命中率**
   - 查看缓存命中情况（目标：>80%）

## 📊 费用说明

### 计费项

1. **流量费用**（后付费）
   - 中国大陆流量：¥0.24/GB
   - 预计月费用：取决于访问量

2. **请求次数费用**（后付费）
   - 中国大陆请求：¥0.15/万次
   - 预计月费用：取决于访问量

### 成本优化建议

- 开启 **缓存过期时间**：减少回源流量
- 使用 **流量包**：如果流量较大，购买月/年流量包更优惠
- 预估费用：如果日访问 1000 人，月费用约 ¥10-50

## ⚠️ 常见问题

### Q1: 配置后访问显示 404？

**可能原因**：
- DNS 未完全生效（等待 10-30 分钟）
- CDN 缓存了错误内容
- 源站配置错误

**解决方法**：
- 清除 CDN 缓存：在 CDN 控制台 → **刷新缓存**
- 检查源站地址是否正确

### Q2: HTTPS 证书配置失败？

**解决方法**：
- 确保域名已正确解析到腾讯云 CDN
- 使用腾讯云 DNS 可以一键申请免费证书
- 或使用 Let's Encrypt 手动上传证书

### Q3: 如何禁用 CDN？（回滚到直接访问 Vercel）

**方法 1**：删除 DNS 的 CNAME 记录
**方法 2**：在 CDN 控制台 → **域名管理** → **禁用域名**
