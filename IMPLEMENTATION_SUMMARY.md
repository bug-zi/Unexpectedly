## 🔧 技术实现

### 核心依赖

```json
{
  "html2canvas": "^1.4.1",    // DOM 转图片
  "jspdf": "^2.5.1",          // PDF 生成
  "date-fns": "^3.3.1"        // 日期格式化
}
```

### 关键技术点

1. **html2canvas**
   - 将 DOM 元素渲染为 Canvas
   - 支持高分辨率输出
   - 处理跨域资源

2. **jsPDF**
   - 生成 PDF 文档
   - 自动分页处理
   - 图片嵌入

3. **React Hooks**
   - useState: 管理对话框状态
   - useRef: 获取 DOM 引用
   - useMemo: 缓存计算结果

4. **Framer Motion**
   - 对话框动画
   - 进度条动画
   - 页面过渡效果

## 📝 使用流程

### 单个回答导出

1. 用户在问题页面回答问题
2. 点击"保存"按钮
3. 自动弹出导出对话框
4. 选择导出格式
5. 点击导出按钮
6. 文件下载到本地

### 批量导出

1. 进入成长跟踪页面
2. 选择时间范围
3. 点击"导出数据"按钮
4. 选择导出格式
5. 等待导出完成
6. 多个文件下载到本地

### 社交分享

1. 在导出对话框中选择平台
2. 点击复制按钮
3. 粘贴到目标平台
4. 配合导出的图片发布

## 🎨 设计特点

### 卡片设计

- **顶部装饰条**：根据问题分类显示不同颜色
- **分类标签**：显示问题分类和图标
- **时间戳**：显示回答日期
- **问题内容**：大字体显示问题
- **难度星级**：1-5 星显示难度
- **回答内容**：保留原有格式
- **元数据**：思考时间、字数统计
- **标签**：显示问题标签
- **水印**：可选的品牌标识

### 主题支持

- **浅色主题**：白色背景，适合打印
- **深色主题**：深色背景，适合分享

## 🐛 已知问题

1. **PDF 中文字体**
   - 当前使用图片嵌入方式
   - 纯文本 PDF 需要引入中文字体包

2. **批量导出性能**
   - 大量记录可能导致浏览器卡顿
   - 建议分批导出

3. **跨域图片**
   - 某些图片可能无法导出
   - 需要服务器配置 CORS
