## 项目概述

### 项目名称
**万万没想到**

### 一句话描述
一个通过精心设计的问题，引导个人思考、提升思维能力和创造力的日常思维工具。

### 核心价值
- 🎲 **随机性**: 永远不知道下一个问题是什么
- 🧠 **启发性**: 问题本身就有洞察力
- 💡 **意外感**: 答案往往"万万没想到"
- 📈 **成长性**: 记录思维进化的轨迹

### 目标用户
- 希望提升思维能力的大学生/职场人
- 创意工作者（设计师、写作者、产品经理等）
- 喜欢自我反思和深度思考的个人

## 产品定位

### 当前问题诊断
```
❌ 偏离初衷：过度强调3D视觉、时光胶囊等"工具"属性
✅ 应该聚焦：**问题本身**和**思考过程**
```

### 产品定位陈述
**"万万没想到"不是一个时光管理工具，而是一个每天5分钟的思维健身房。通过精心设计的问题，让用户在回答中发现自己的思维盲区，收获'万万没想到'的洞察。**

### 差异化特色
| 维度 | 其他产品 | 万万没想到 |
|------|---------|-----------|
| 问题来源 | 固定题库 | ✅ 双维分类系统 + 纯随机 |
| 回答方式 | 只能文字 | ✅ 多种思考模式 |
| 反馈机制 | 无反馈 | ✅ 7天后对比思维变化 |
| 核心玩法 | 打卡签到 | ✅ 灵感老虎机 |
| 激励机制 | 天数累计 | ✅ 彩蛋 + 思维可视化 |

## 问题分类系统

### 双层分类结构

#### 第一层：主分类
```
┌──────────┬──────────┬──────────┐
│   🧠     │   🌍     │   🎲     │
│思维维度  │生活场景  │纯随机    │
└──────────┴──────────┴──────────┘
```

#### 第二层：思维维度细分（5类）

**1. 🔮 假设思维**
- 核心问题：如果...会怎样？
- 训练目标：想象力、场景构建能力
- 示例问题：
  - 如果金钱不是问题，你在做什么？
  - 如果今天是你生命的最后一天，你会做什么？
  - 如果你可以改变过去的一个决定，会是哪个？

**2. 🔄 逆向思考**
- 核心问题：如何故意毁掉...
- 训练目标：批判性思维、问题发现能力
- 示例问题：
  - 如何故意毁掉你的人际关系？
  - 如何故意让一个项目失败？
  - 如何故意毁掉你的健康？

**3. 🔗 联想创意**
- 核心问题：A + B 会产生什么？
- 训练目标：创新思维、跨界整合能力
- 示例问题：
  - 咖啡 + 书店 = ？
  - 音乐 + 建筑 = ？
  - 失败 + 成功 = ？

**4. 💡 自我反思**
- 核心问题：你的真实想法是...
- 训练目标：自我认知、价值观澄清
- 示例问题：
  - 你最近一次感到骄傲是什么时候？
  - 你最害怕别人发现你的什么特质？
  - 什么事让你觉得"这才是真正的我"？

**5. 🚀 未来设想**
- 核心问题：10年后...
- 训练目标：愿景规划、目标设定
- 示例问题：
  - 10年后你希望自己有什么改变？
  - 如果你能给5年后的自己一句话，会是什么？
  - 理想情况下，5年后的你在做什么？

#### 第二层：生活场景细分（5类）

**1. 💼 职业发展**
- 职业规划
- 工作技能
- 团队协作
- 创业想法

**2. 🎨 创意激发**
- 创意训练
- 表达方式
- 灵感记录
- 创作习惯

**3. ❤️ 人际关系**
- 沟通方式
- 亲密关系
- 社交恐惧
- 价值边界

**4. 📚 学习成长**
- 学习方法
- 知识体系
- 技能提升
- 阅读习惯

**5. 🌱 生活哲学**
- 人生意义
- 价值观
- 时间管理
- 生活平衡

## 灵感老虎机

### 功能概述
通过老虎机形式随机生成三个词语，用户基于这三个词语展开联想，写一段话。这是一个创意训练的附加玩法。

### 交互流程
```
用户点击"灵感老虎机"
    ↓
进入老虎机页面
    ↓
点击"🎲 抽取灵感"
    ↓
老虎机动画启动（3个滚轮）
    ↓
逐个停下（延迟效果）
    ↓
显示3个词语
    ↓
提示用户联想写作
    ↓
用户输入联想内容
    ↓
保存到历史记录
```


### 核心算法

#### 问题推荐算法
```typescript
function recommendQuestion(
  userHistory: Answer[],
  questionPool: Question[]
): Question {
  // 1. 分析用户偏好
  const categoryPreference = analyzeCategoryPreference(userHistory);

  // 2. 避免重复（30天内未回答过）
  const recentQuestionIds = getRecentQuestionIds(userHistory, 30);

  // 3. 根据偏好选择类别
  const preferredCategory = selectCategory(categoryPreference);

  // 4. 从该类别中筛选未回答问题
  const candidates = questionPool.filter(q =>
    q.category.secondary === preferredCategory &&
    !recentQuestionIds.includes(q.id)
  );

  // 5. 随机选择
  return candidates[Math.floor(Math.random() * candidates.length)];
}
```

#### 思维变化检测
```typescript
function detectThoughtChange(
  oldAnswer: Answer,
  newAnswer: Answer
): ThoughtChange {
  // 1. 关键词提取
  const oldKeywords = extractKeywords(oldAnswer.content);
  const newKeywords = extractKeywords(newAnswer.content);

  // 2. 对比关键词
  const keywordChanges = {
    added: newKeywords.filter(k => !oldKeywords.includes(k)),
    removed: oldKeywords.filter(k => !newKeywords.includes(k)),
    persistent: oldKeywords.filter(k => newKeywords.includes(k))
  };

  // 3. 情绪分析
  const moodChange = {
    from: analyzeMood(oldAnswer.content),
    to: analyzeMood(newAnswer.content),
    confidence: calculateConfidence(oldAnswer, newAnswer)
  };

  // 4. 深度分析
  const depthChange = {
    from: analyzeDepth(oldAnswer.content),
    to: analyzeDepth(newAnswer.content),
    evidence: collectEvidence(oldAnswer, newAnswer)
  };

  return { keywordChanges, moodChange, depthChange };
}
```

#### 老虎机词语生成
```typescript
function generateSlotMachineWords(): [string, string, string] {
  const categories = Object.keys(wordCategories);

  // 策略1: 完全随机
  if (Math.random() < 0.7) {
    return [
      getRandomWord(wordCategories),
      getRandomWord(wordCategories),
      getRandomWord(wordCategories)
    ];
  }

  // 策略2: 同类别（增加趣味性）
  const category = categories[Math.floor(Math.random() * categories.length)];
  return [
    getRandomWord(wordCategories[category]),
    getRandomWord(wordCategories[category]),
    getRandomWord(wordCategories[category])
  ];
}
```
